# Profession System

Gathering, crafting, and secondary professions with skill progression and recipe acquisition.

## Overview

- Each character can have **2 primary professions** + **all secondary professions**
- Profession skill ranges from **1 to 300**
- Skill increases by crafting/gathering appropriate-level items
- Higher skill unlocks better recipes

## Gathering Professions (3)

| Profession | Description | Feeds Into |
|-----------|-------------|------------|
| Mining | Gather ore from resource nodes | Blacksmithing, Engineering |
| Herbalism | Collect herbs | Alchemy |
| Skinning | Harvest leather from defeated beasts | Leatherworking |

## Crafting Professions (6)

| Profession | Description |
|-----------|-------------|
| Blacksmithing | Craft plate armor and weapons |
| Leatherworking | Craft leather and mail armor |
| Tailoring | Craft cloth armor |
| Alchemy | Create potions and elixirs for buffs |
| Enchanting | Add permanent stat bonuses to gear |
| Engineering | Create gadgets, bombs, and unique items |

## Secondary Professions (3)

All characters can learn all secondary professions (these do not count toward the 2-primary limit):

- **Cooking**
- **First Aid**
- **Fishing**

## Recipe Acquisition

Recipes are acquired from multiple sources:

- Class trainers (baseline recipes at skill thresholds)
- World drops from monsters
- Reputation vendors (at specific faction tiers)

## Profession Synergies

Alt characters are encouraged to take different profession pairs to create synergies across the account (see [prestige.md](prestige.md) for alt system details).
