# Prestige and Meta-Progression

Ascension system, Paragon talent trees, and alt character system.

## Ascension System (Primary Prestige)

### Trigger

After reaching level 60 and clearing The Eternal Crypt raid, players can "Ascend" their character, gaining permanent account-wide bonuses.

### Benefits Per Ascension

Each Ascension grants:

- +2% XP gain (account-wide)
- +1% gold gain (account-wide)
- +1% gear drop chance (account-wide)
- 1 Ascension Point for Paragon talent tree
- Prestige cosmetic rewards (titles, transmog appearances)

### What Resets

Character resets to level 1 with:

- Level reset to 1
- Gear removed (except heirlooms)
- Quest progress reset
- Zone progress reset

### What Is Retained

- All unlocked races and classes
- Profession knowledge (recipes remain learned)
- Achievement progress
- Mount and pet collections
- Transmog (appearance) collection
- Account-wide Paragon bonuses

## Paragon System

Ascension Points are spent in account-wide Paragon talent trees:

| Tree | Max Bonus | Description |
|------|----------|-------------|
| Power | +50% | Increase damage and healing |
| Resilience | +50% | Increase health and survivability |
| Fortune | +30% | Increase loot quality and gold drops |
| Swiftness | +40% | Increase leveling speed and quest completion |
| Mastery | Special | Unlock special abilities and convenience features |

- Each tree contains **50 talent nodes**
- Requires **250 total Ascensions** to max all trees

## Alt Character System

### Benefits of Alts

- Experience different class/race combinations
- All alts benefit from account-wide Paragon bonuses
- Shared gold and currency pools
- Profession synergies (alts can have different profession pairs)
- Achievement hunting across multiple characters

### Alt-Specific Advantages

- Reduced leveling time due to Paragon bonuses
- Account-wide reputation gains at 50% rate
- Heirloom gear system (bind-on-account items that scale with level)
